## 1. Introduction

### 1.1 Positionnement

**GLOBAL / Apogée** est la **plateforme digitale officielle du réseau HelpConfort**, conçue pour centraliser, simplifier et optimiser la gestion quotidienne des agences de services à domicile.

Plus qu'un simple outil, c'est un **écosystème complet** qui accompagne chaque collaborateur dans ses missions : de l'assistante qui recherche une procédure, au dirigeant qui pilote son activité, jusqu'à la tête de réseau qui supervise l'ensemble des franchises.

### 1.2 À qui s'adresse cette solution ?

| Profil | Bénéfices clés |
|--------|----------------|
| **Dirigeants d'agence** | Pilotage temps réel, KPIs, prise de décision éclairée |
| **Assistantes** | Documentation centralisée, support rapide, gain de temps |
| **Commerciaux** | Guide apporteurs, processus standardisés |
| **Techniciens** | Accès mobile aux procédures, planning |
| **Tête de réseau** | Vision consolidée, benchmark, redevances |

### 1.3 Problématiques résolues

❌ **Avant** : Documentation éparpillée (mails, fichiers, classeurs)  
✅ **Après** : Base de connaissance unique et à jour

❌ **Avant** : Questions répétitives = temps perdu  
✅ **Après** : Assistant IA disponible 24/7

❌ **Avant** : Pilotage à l'aveugle (tableaux Excel manuels)  
✅ **Après** : Indicateurs temps réel depuis Apogée

❌ **Avant** : Support par email = délais, perte d'historique  
✅ **Après** : Chat + tickets traçables avec SLA

❌ **Avant** : Aucune vision réseau pour la franchise  
✅ **Après** : Dashboard multi-agences consolidé

## 3. Modules Clés du SaaS

### 3.1 Help! Academy — Centre de Documentation

> *"Toute la connaissance métier à portée de clic"*

#### Objectif métier
Centraliser et maintenir à jour l'ensemble de la documentation opérationnelle du réseau.

#### Fonctionnalités

- **Guide Apogée** : Documentation complète du CRM (clients, dossiers, devis, factures, interventions, planning)
- **Guide Apporteurs** : Processus de gestion des apporteurs d'affaires
- **Base Documentaire** : Procédures internes, modèles, documents officiels

#### Bénéfices

✅ Information à jour pour tous  
✅ Formation autonome des équipes  
✅ Réduction des erreurs opérationnelles  
✅ Système de favoris personnalisé

#### Parcours utilisateur

```
Connexion → Help! Academy → Sélection du guide
    → Navigation par catégorie → Consultation de la section
    → Ajout aux favoris pour accès rapide
```

---

### 3.2 Assistant IA — Mme MICHU

> *"Votre experte Apogée disponible 24h/24"*

#### Objectif métier
Répondre instantanément aux questions métier sans mobiliser de ressources humaines.

#### Fonctionnalités

- Chat conversationnel en langage naturel
- Réponses basées sur la documentation officielle
- Citations des sources pour vérification
- Escalade vers conseiller humain si nécessaire
- Historique des conversations

#### Bénéfices

✅ Réponses immédiates (vs attente support)  
✅ Disponibilité permanente  
✅ Désengorgement du support  
✅ Montée en compétence des équipes

#### Exemple d'interaction

```
Utilisateur : "Comment créer un devis multi-univers dans Apogée ?"

Mme MICHU : "Pour créer un devis multi-univers :
1. Ouvrez le dossier client
2. Cliquez sur 'Nouveau devis'
3. Sélectionnez les univers concernés...
[Voir le guide complet : Guide Apogée > Devis > Multi-univers]"
```

### 3.3 Pilotage Agence — Indicateurs en Temps Réel

> *"Pilotez votre agence avec des données fiables"*

#### Objectif métier
Fournir aux dirigeants une vision claire et actualisée de l'activité de leur agence.

#### Fonctionnalités

| Indicateur | Description |
|------------|-------------|
| **CA Global** | Chiffre d'affaires facturé (jour/mois/année) |
| **Dossiers** | Nombre créés, clôturés, en cours |
| **Transformation** | Taux devis → facture |
| **SAV** | Taux, délais moyens |
| **Par apporteur** | CA et dossiers par source d'affaires |
| **Par univers** | Répartition plomberie/électricité/etc. |
| **Par technicien** | Productivité individuelle |

#### Bénéfices

✅ Décisions basées sur des faits  
✅ Identification rapide des alertes  
✅ Objectifs suivis en temps réel  
✅ Plus de reporting manuel Excel

#### Mode Diffusion TV

Affichez vos KPIs sur un écran en agence pour motiver les équipes !

## 5. Cas d'Usage Concrets

### 5.1 Pour une assistante

**Situation** : Une assistante doit créer un avoir sur une facture mais ne se souvient plus de la procédure.

**Avant** : Appel à un collègue, recherche dans les emails, 15 minutes perdues.

**Avec GLOBAL/Apogée** :
1. Ouvre le chat IA
2. Tape : "Comment faire un avoir sur facture ?"
3. Reçoit la procédure détaillée en 10 secondes
4. Applique les étapes

**Gain** : 14 minutes économisées, collègue non dérangé.
